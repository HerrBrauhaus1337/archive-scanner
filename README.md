# archive-scanner
Script to filter the e-mailaddresses from a 7z-archive

## Dependencies
- WSL with Ubuntu
  - p7zip-full
  - jq
  - pv
  - grep
  - gawk
  - coreutils
 
## Installation

- sudo apt update
- sudo apt upgrade
- sudo apt install p7zip-full jq pv grep gawk coreutils

## Usage
./scan.sh
